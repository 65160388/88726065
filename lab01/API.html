Slot API

27 สิงหาคม 2566

สารบัญบทความแอนดรอยด์ ถึงแม้ว่าจะเรียกว่า Slot API ก็ตาม แต่ก็ไม่ได้หมายถึง API ที่มีให้เรียกใช้งานใน Jetpack Compose หรอกนะ เพราะ Slot API เป็น Compose Pattern ที่จะช่วยให้นักพัฒนาสามารถสร้าง Reusable Composable ให้นำไปใช้งานได้สะดวกมากขึ้น
